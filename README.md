# MLEssential
MLEssential is a computational toolbox to predict essential genes for yeast species using machine learning based on multi-scale information, including sequence features and evolution-based features.

## Dependencies
* Python==3.7.4
* biopython==1.75
* scikit-learn==0.22.1
* numpy==1.17.2
* scipy==1.3.1
* pandas==0.25.1
* seaborn==0.9.0
